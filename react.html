📌 What is React?

React is a JavaScript library used to build User Interfaces (UI).

It is mainly used for building Single Page Applications (SPA) where content changes without reloading the page.

React was created by Jordan Walke in 2011 and officially released in 2013 by Meta (Facebook).

✅ Why do we use React?

  • Fast Performance – Updates only the required parts of the page.
  • Reusable Components – Small components can be reused.
  • Easy to Manage – Code is clean and well organized.
  • Popular – Used by big companies like Meta, Netflix, Instagram.

✅ Features of React

  • JSX – JavaScript + HTML syntax
  • Component-Based Architecture
  • One-Way Data Binding
  • Virtual DOM (makes React fast)
  • Reusable Components

✅ Library vs Framework

  • React is a Library → It focuses only on UI.
  • Framework → Provides full structure (Example: Angular).

✅ NPM vs NPX

  • NPM → Used to install and manage packages.
  • NPX → Runs packages directly without installing.

Example:

npm install react
npx create-react-app myapp

✅ What is State in React?

State is internal data of a component that can change over time.

When state changes, React automatically updates the UI.

Examples:

  • Counter value
  • Form input data
  • Toggle status
